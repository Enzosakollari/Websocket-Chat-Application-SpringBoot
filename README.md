## ğŸ’¬ Overview

This is a classic **Spring Boot WebSocket Chat Application** designed to demonstrate real-time messaging using WebSockets. The application includes both a simple backend (built using Spring Boot) and a frontend (using HTML, CSS, and JavaScript) that allows users to chat in real-time. The data is stored and managed in a MongoDB database, and the application uses WebSocket for seamless communication.

---

## ğŸ”‘ Features

- **Real-Time Messaging**: Communication through WebSockets enables instant updates between users.
- **MongoDB**: The app stores chat messages and user data in a MongoDB database for persistence.
- **Simple and Clean UI**: The frontend is simple, with a user-friendly chat interface built using HTML, CSS, and JS.
- **Scalable**: Designed to handle multiple chat rooms and users simultaneously.
